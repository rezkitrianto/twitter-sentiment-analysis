REQUIREMENTS:

- lmdb (to install lmdb, run the following command in the terminal: pip install lmdb)
- caffe (consult the following website to install caffe -> http://caffe.berkeleyvision.org/installation.html)

RUN EXPERIMENTS:
1. Change directory to the directory where the run-1-domain.sh located.

3. run in the terminal: "./run-all-domain.sh" to run the case of all domain
